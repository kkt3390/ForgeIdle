# ForgeIdle 패치노트

## 2026-06-01

### 골드 보상과 강화 이력 개선

- 모든 사냥터의 골드 획득량을 기존 대비 5배로 높였습니다.
- 개인별 최근 기록을 최대 100줄까지 유지하도록 변경했습니다.
- 모든 강화 시도의 적용 확률, 난수, 보호권 사용 여부와 결과를 별도 테이블에 기록합니다.
- SSMS에서 강화 단계별 실제 확률을 집계할 수 있는 조회 SQL을 추가했습니다.
- 운영 환경에서는 테스트 로그인 기능이 항상 차단되도록 변경했습니다.

## 2026-05-31

### 닉네임, 랭킹, 테스트 로그인 추가

- 최초 로그인 시 닉네임을 설정하도록 변경했습니다.
- 레벨, 현재 강화도, 최고 강화도를 확인할 수 있는 실시간 랭킹을 추가했습니다.
- 로그인하지 않은 화면에서는 로그아웃 버튼이 보이지 않도록 수정했습니다.
- 기능 점검용 운영자 테스트 로그인을 추가했습니다.
- 네이버 로그인을 제거하고 카카오 로그인만 유지했습니다.

### 운영 인증 안정화

- 서버가 재시작되거나 다시 배포되어도 인증 키가 유지되도록 변경했습니다.
- 소셜 로그인 실패 시 사용자에게 안내 메시지를 표시하도록 수정했습니다.

### 자동 배포 구성

- GitHub `main` 브랜치에 변경 사항이 올라오면 MonsterASP.NET으로 자동 배포되도록 구성했습니다.
