# 몬스터 도감 이미지

도감 이미지는 아래 파일명 규칙으로 넣습니다.

```text
area-00-normal-01.webp
area-00-elite-01.webp
area-00-golden-01.webp
```

- `area-00`은 사냥터 번호입니다.
- 등급은 `normal`, `elite`, `golden`을 사용합니다.
- 마지막 번호는 등급별 `01`부터 `10`까지입니다.
- 원본이 `2048x2048`이면 비율을 유지한 `512x512 WebP` 썸네일로 변환해 올립니다.
- 이미지가 아직 없으면 화면에는 미등록 자리만 표시됩니다.
